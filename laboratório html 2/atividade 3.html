Listas de definição

HTML
Linguagem de marcação de hipertexto é utilidada na construção de
páginas na Web. Documentos HTML podem ser interpretados por
navegadores. A tecnologia é fruto da junção entre os HyTime e
SGML.
CSS
Cascading Style Sheets é um mecanismo para adicionar estilo
a um documento web.
